# 7 Seas Brewing — Loyalty App Demo

## Quick Start
```bash
npm install
npm run dev
```

## Deploy to Vercel
```bash
npm i -g vercel
vercel
```
Or connect the repo to Vercel and it auto-detects Vite.

## Structure
- `src/App.jsx` — Full brewery loyalty app (consumer + admin)
- `src/DemoWrapper.jsx` — Sales presentation wrapper
- `src/demo.config.js` — Wrapper configuration

## Stack
- Vite + React 18 + Lucide React
